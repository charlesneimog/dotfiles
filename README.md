# Installation

To install these dotfiles on your system, follow these steps:

Clone this repository to your local machine:

``` bash
mkdir -p Git
git clone https://github.com/charlesneimog/dotfiles.git ~/Documents/Git/
cd ~/Documents/Git/dotfiles
source ./installer.sh
```
